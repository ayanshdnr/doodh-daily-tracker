# Doodh Daily Tracker
A simple milk tracking application for recording daily milk collection, expenses, and reports.
## Features

- Daily milk entry
- Customer management
- Expense tracking
- PDF export
- Excel export
- Local data storage
## Installation

```bash
npm install
npm run dev
